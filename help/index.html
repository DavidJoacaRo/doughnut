Doughnut Commands


General

dn!help - Redirects you to this page.
dn!help music - Redirects you to this page. (again)
dn!credits - Credits to contributors and collaborators.
dn!donate - Simply gives a donation link.
dn!github - Gives a link to the bot's repository



Image

dn!animeme - Sends a random post from r/animemes
dn!aww - Sends a random post from r/aww
dn!comedyheaven - Sends a random post from r/comedyheaven
dn!4chan - Sends a random greentext
dn!requestavatar (@mention) - Generates the mentioned's user avatar
dn!meme - Sends a random meme from random meme subreddits.
dn!sbubby - Sends a random post from r/sbubby
dn!bonk - wait, it's this supposed to be here?
dn!wikihow - how to make friends, oh wait this is not google?



Music

dn!play (link only) - Plays the specified link. (If more songs are added at the same time it's gonna make a queue).
dn!stop - Stops playing and leaves the voice channel.
dn!skip - Skips and plays the next music in the queue.
dn!queue - Shows the server queue.
dn!lyirics [Song Name] - Shows the lyrics of the specified song.
